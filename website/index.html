Built for local control

Not a chatbot. A foundation for privacy-sensitive agent pipelines with policy, audit, and cryptographic receipts.

  • Sovereign by default

    Your proving keys, your managed/ tree, your machine. No cloud trust boundary and no telemetry.

  • Living governance

    Agents read rules, workflows, and knowledge as markdown at runtime. Policy changes use propose → approve → apply.

  • Proof receipts

    Dual-track Groth16 (governance + entity) over BN254, plus optional Proof Profile claims (catalog predicates — not free-form circuits).

  • Studio + Workbench

    Author Agents, Packs, and Proof Profiles; promote to the board; run governed pipelines without editing repository files.

How it works

A single honest path from bootstrap to verified artifact.

  1. 01

    Bootstrap

    Sovereign setup generates operator-owned ZK keys and records provenance in install.json.

  2. 02

    Author

    Studio: Save → Test → Promote Agents, Packs, and Proof Profiles. Rules stay living markdown under signed capabilities.

  3. 03

    Run

    Workbench: assemble from the shelf, bind a proof profile if you want a claim, Run. Artifacts land in SQLite + CAS.

  4. 04

    Verify

    Trust hub: check claims and dual-track proofs with your keys. Export a verifier bundle for third parties.

Platform surface

What ships in the open-source runtime today.

  • Desktop + operator UI

    Windows and Linux installers. Studio, Workbench, Runs, Artifacts, Trust, System, Settings.

  • Local API v2

    /api/v2/* for packs, pipelines, jobs SSE, governance, and system health.

  • Redaction engine v3

    Format-aware pattern redaction with structured entities[] for entity proofs.

  • Chained audit

    Hash-linked logs for system and agents. Doctor and integrity repair for operators.

  • Dual-track ZK

    3 governance circuits + 6 default entity circuits. Optional voice redaction path.

  • Bring your own LLM

    Pluggable backends (Ollama example). No bundled model weights.

Includes

  • Sovereign bootstrap (desktop, Docker, native)
  • Studio (Agents, Packs, Proof Profiles) + Workbench
  • Three official agent packs
  • Doctor, integrity checks, install scripts

Does not include

  • HIPAA / SOC2 / GDPR certification
  • Enterprise DLP-grade redaction
  • Cloud SaaS or multi-tenant hosting
  • Proof that an LLM answer was “correct”
  • Arbitrary browser-authored ZK circuits
